Jika Anda mengklik tombol "Kirim", data formulir akan dikirim ke halaman bernama"/coba_form.html"

Jika menggunakan target _self akan berada di tab yang sama, jika menggunakan _blank akan membuat tab baru

Jika menggunakan method GET di url akan menampilkan parameteri_name=tria&i_belakang=ajeng, jika menggunakan method POST data form tidak akan dikirim pada URL melainkan pada body request
